2261. K Divisible Elements Subarrays

зөҰе®ҡдёҖеҖӢзҹ©йҷЈпјҢи·ҹе…©еҖӢж•ёеӯ—kи·ҹpпјҢжұӮдёҚзӣёзӯүзҡ„subarrayдёҰдё”зҹ©йҷЈе…§еҸҜиў«pж•ҙйҷӨзҡ„ж•ёйҮҸдёҚи¶…йҒҺkеҖӢзҡ„ж•ёзӣ®гҖӮ

йӣҷеұӨиҝҙеңҲ+hash

йҖҷйЎҢжҲ‘еҖ‘йЎҢзӣ®дё»иҰҒзҡ„йҮҚй»һеңЁж–јеҰӮдҪ•еҲӨж–·е…©еҖӢsubarrayжҳҜеҗҰзӣёзӯүпјҢиҖҢж•ҙйҷӨзҡ„йғЁеҲҶпјҢжҲ‘еҖ‘еҸӘйңҖиҰҒиЁӯе®ҡдёҖеҖӢеҲӨе®ҡејҸpruneе°ұеҸҜд»Ҙ

жңҖзӣҙжҺҘзҡ„еҒҡжі•жҳҜпјҢжҲ‘еҖ‘з”Ёе…©еұӨиҝҙеңҲпјҢеҺ»еҲӨж–·жүҖжңүsubarrayпјҢеҰӮжһңи¶…йҒҺж•ҙйҷӨеҖӢж•ёе°ұcontinueпјҢз„¶еҫҢйҖҸйҒҺдёҖеҖӢsetдҫҶеӯҳж”ҫзҹ©йҷЈ

йҖҷйӮҠзүҪжүҜеҲ°иҰҒеҰӮдҪ•hashдёҖеҖӢsubarray

еңЁpythonдёӯпјҢlistжҳҜmutableпјҢжүҖд»ҘжҳҜinhashableпјҢдёҚиғҪзӣҙжҺҘиў«ж”ҫйҖІsetиЈЎйқўпјҢжҲ‘еҖ‘еҝ…й Ҳе°Үе®ғиҪүжҲҗimmutableзҡ„йЎһеҲҘ

дҪҶдҪ жңғзҷјзҸҫпјҢйҖҷйәјеҒҡжҷӮй–“иӨҮйӣңеәҰеҸҜиғҪжңғдҫҶеҲ°$O(n^3)$

class Solution:
    def countDistinct(self, nums: List[int], k: int, p: int) -> int:
        ans = set()
        for i in range(len(nums)):
            cnt = 0
            for j in range(i, len(nums)):
                if nums[j] % p == 0:
                    cnt += 1
                if cnt > k:
                    break
                ans.add(tuple(nums[i:j+1]))
        return len(ans)

Rolling hash

жҲ‘еҖ‘еҸҜд»Ҙз”ЁжӣҙеҘҪзҡ„hash functionпјҢйҖҸйҒҺиЁӯе®ҡдёҖеҖӢBASEи·ҹеӨ§иіӘж•ёMODпјҢз”ұж–јйЎҢзӣ®зөҰе®ҡжҜҸеҖӢж•ёеӯ—е°Ҹж–јзӯүж–ј200

жүҖд»ҘBASEеҸҜд»ҘиЁӯе®ҡзӮә200пјҢз„¶еҫҢMODиЁӯе®ҡдёҖеҖӢеӨ§зҡ„иіӘж•ёпјҢеғҸжҳҜ10**9+7

дҪҶжҳҜйҖҷжЁЈеҸҜиғҪжңғзҷјз”ҹcollisionпјҢиҰҒжүҫеҲ°йҒ©еҗҲдёҚжңғзҷјз”ҹзў°ж’һзҡ„BASEи·ҹMODжңүй»һйӣЈ

Trie

жҲ‘еҖ‘д№ҹеҸҜд»ҘйҖҸйҒҺTrieдҫҶеҲӨж–·жңүжІ’жңүзҷјз”ҹйҮҚиӨҮзҡ„subarrayпјҢеҰӮжһңзҷјз”ҹйҮҚиӨҮзҡ„зӢҖжіҒе°ұи·ійҒҺ

class TrieNode:
    def __init__(self):
        self.children = {}

class Solution:
    def countDistinct(self, nums: List[int], k: int, p: int) -> int:
        root = TrieNode()
        count = 0
        n = len(nums)

        for i in range(n):
            node = root
            div_count = 0
            for j in range(i, n):
                if nums[j] % p == 0:
                    div_count += 1
                if div_count > k:
                    break
                if nums[j] not in node.children:
                    node.children[nums[j]] = TrieNode()
                    count += 1
                node = node.children[nums[j]]
        
        return count
