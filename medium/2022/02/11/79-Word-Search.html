79. Word Search

и§ЈйЎҢжҖқи·Ҝ

йҖҷйЎҢжҲ‘еҖ‘еҫҲзӣҙжҺҘжңғжғіиҰҒз”ЁBFS/DFSзҡ„и§Јжі•дҫҶи§ЈпјҢд»ҘжҜҸеҖӢй»һзӮәиө·й»һеҺ»еҒҡжҗңе°ӢпјҢйҖҷжЁЈжҷӮй–“иӨҮйӣңеәҰзӮә$O(M N 3^L)$пјҢ$M, N$зӮәжЈӢзӣӨзҡ„й•·еҜ¬пјҢ$L$зӮәжҗңе°Ӣеӯ—дёІзҡ„й•·еәҰпјҢжңҖе·®зҡ„жғ…жіҒдҫҶиӘӘпјҢжҲ‘еҖ‘йңҖиҰҒиө°иЁӘ$M \times N$еҖӢй»һдҪңзӮәиө·й»һпјҢжҜҸеҖӢй»һеҸҲйңҖиҰҒиө°йҒҺ$L$еҖӢй»һеҒҡжҺўзҙўпјҢжҜҸеҖӢдҪҚзҪ®еҸҜд»ҘеҫҖдёҠдёӢе·ҰеҸіиө°пјҢжүҖд»ҘжңҖеЈһжңү$4*3^{L-1}$зЁ®еҸҜиғҪгҖӮ

йҢҜиӘӨжҖқи·Ҝ

жҲ‘еҖ‘еҸҜд»Ҙи©Ұи‘—жғідёҖдёӢжңүд»Җйәјең°ж–№жҳҜйҮҚиӨҮзҡ„иЁҲз®—пјҢдҪҶе…Ҳи·ҹеӨ§е®¶иӘӘдёӢйқўзҡ„жғіжі•жҳҜйҢҜзҡ„

еҒҮиЁӯжҲ‘еҖ‘зҡ„жЈӢзӣӨжҳҜ

$ABEOK$ $CAOLL$

иҖҢжҲ‘еҖ‘иҰҒжҗңзҙўзҡ„еӯ—дёІзӮә$ABEOP$пјҢеҒҮиЁӯжҲ‘еҖ‘жҳҜз”ЁDFSпјҢеҸҜд»ҘзңӢеҲ°жҲ‘еҖ‘еҫһз¬¬дёҖеҖӢ$A (0,0)$еҮәзҷјеҫҢпјҢжңғи©ҰйҒҺпјҡ

$A\rightarrow B\rightarrow E\rightarrow O\rightarrow K\rightarrow Fail$

$A\rightarrow B\rightarrow E\rightarrow O\rightarrow L \rightarrow Fail$

д№ӢеҫҢжҲ‘еҖ‘еҲ°дёӢдёҖеҖӢ$A(1,1)$пјҢд№ҹжңғеҺ»жҗңе°Ӣпјҡ

$A\rightarrow B\rightarrow E\rightarrow O\rightarrow K\rightarrow Fail$

$A\rightarrow B\rightarrow E\rightarrow O\rightarrow L \rightarrow Fail$

еҸҜд»ҘзңӢеҲ°йҖҷе…©еҖӢи·Ҝеҫ‘йҖҡйҒҺеҗҢдёҖеҖӢй»һ$B(0,1)$пјҢд№ӢеҫҢзҡ„и·Ҝеҫ‘жҳҜдёҖжЁЎдёҖжЁЈзҡ„пјҢжүҖд»Ҙз•¶жҲ‘еҖ‘з¬¬дёҖж¬ЎжҗңзҙўзҹҘйҒ“еҫҢйқўе·Із¶“жүҫдёҚеҲ°жҷӮпјҢз¬¬дәҢж¬ЎжҗңзҙўжҮүи©Іе°ұеҸҜд»ҘзӣҙжҺҘеҒңжӯўпјҡ

$A\rightarrow B\rightarrow Fail$

$A\rightarrow B\rightarrow Fail$

еӨ§е®¶жңүзҷјзҸҫйҖҷеҖӢжғіжі•йҢҜиӘӨеңЁе“Әе—Һпјҹ

еӣ зӮәжҲ‘еҖ‘е…©ж¬Ўиө°еҲ°еҗҢеҖӢй»һ$B(0,1)$зҡ„жҷӮеҖҷпјҢжЈӢзӣӨзҡ„иө°иЁӘзӢҖж…ӢдёҰдёҚеҗҢпјҢжүҖд»Ҙз¬¬дёҖж¬ЎжҺўзҙўзҡ„зөҗжһңпјҢдёҰдёҚиғҪзӣҙжҺҘз”ЁеңЁз¬¬дәҢж¬ЎпјҢиҲүдёҖеҖӢдҫӢеӯҗпјҢеҒҮиЁӯжҲ‘еҖ‘д»ҠеӨ©жғіиҰҒжҗңзҙўзҡ„еӯ—дёІзӮә$ABEOA$пјҢйҖҷжЁЈеҫһ$A(1,1)$еҮәзҷјпјҢз¶“йҒҺ$B(0,1)$жҳҜFalseпјҢдҪҶеҫһ$A (0,0)$еҮәзҷјпјҢеҚ»еҸҜд»ҘжҳҜTrueпјҢжүҖд»ҘйҖҷеҖӢж–№жі•дёҰдёҚиғҪйҒ©з”ЁгҖӮ

и§ЈйЎҢжҖқи·Ҝ

еҹәжң¬дёҠзҡ„и§Јжі•жҳҜжҺЎеҸ–DFSпјҢйҷӨжӯӨд№ӢеӨ–жҲ‘еҖ‘иҰҒиЁҳйҢ„жЈӢзӣӨжҳҜеҗҰиө°иЁӘйҒҺзҡ„зӢҖж…ӢпјҢжҲ‘еҖ‘еҸҜд»ҘйҖҸйҒҺдёҖеҖӢдәҢз¶ӯзҹ©йҷЈvisitedдҫҶзҙҖйҢ„пјҢд№ҹеҸҜд»ҘзӣҙжҺҘе°Үз•¶еүҚдҪҚзҪ®дҝ®ж”№жҲҗдёҖеҖӢйқһеӨ§еҜ«еӯ—жҜҚзҡ„еӯ—е…ғ(еӣ зӮәйЎҢзӣ®жңүиЁ»жҳҺжЈӢзӣӨеҸӘжңғжҳҜеӨ§еҜ«еӯ—жҜҚ)гҖӮ

еҜҰдҪң

class Solution:
    def exist(self, board: List[List[str]], word: str) -> bool:
        for i in range(len(board)):
            for j in range(len(board[0])):
                if self.dfs(board, i, j, word):
                    return True
        return False
        
    def dfs(self, board, i, j, word):
        if board[i][j] == "#" or board[i][j] != word[0]:
            return False
        if board[i][j] == word[0] and len(word) == 1:
            return True
        
        tmp = board[i][j]
        board[i][j] = "#"
        if i-1 >= 0 and self.dfs(board, i-1, j, word[1:]):
            return True
        if j+1 < len(board[0]) and self.dfs(board, i, j+1, word[1:]):
            return True
        if i+1 < len(board) and self.dfs(board, i+1, j, word[1:]):
            return True
        if j-1 >= 0 and self.dfs(board, i, j-1, word[1:]):
            return True
        board[i][j] = tmp
        return False
