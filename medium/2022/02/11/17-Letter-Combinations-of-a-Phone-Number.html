17. Letter Combinations of a Phone Number

и§ЈйЎҢжҖқи·Ҝ

йҖҷеҖӢйЎҢзӣ®иҰҒжұӮжҲ‘еҖ‘зөҰе®ҡдёҖеҖӢз”ұж•ёеӯ—зө„жҲҗзҡ„еӯ—дёІпјҢдёҰдё”ж №ж“ҡжүӢж©ҹйҚөзӣӨдёҠзҡ„еӯ—жҜҚе°ҚжҮүй—ңдҝӮпјҢжүҫеҮәжүҖжңүеҸҜиғҪзҡ„еӯ—жҜҚзө„еҗҲгҖӮ

жҲ‘еҖ‘дҪҝз”ЁдёҖеҖӢйҒһиҝҙзҡ„ж–№жі•дҫҶеҜҰзҸҫйҖҷеҖӢйңҖжұӮгҖӮжҲ‘еҖ‘жңғе»әз«ӢдёҖеҖӢз©әзҡ„еҲ—иЎЁеӯҳж”ҫжүҖжңүеҸҜиғҪзҡ„еӯ—жҜҚзө„еҗҲпјҢе’ҢдёҖеҖӢз©әеӯ—дёІдҪңзӮәдёӯй–“йҒҺзЁӢзҡ„жҡ«еӯҳи®ҠйҮҸгҖӮжҲ‘еҖ‘жңғйҒһиҝҙиҷ•зҗҶжҜҸеҖӢж•ёеӯ—пјҢжҜҸеҖӢж•ёеӯ—йғҪжңү3~4зЁ®еҸҜиғҪзҡ„е°ҚжҮүеӯ—жҜҚпјҢжҲ‘еҖ‘жңғжһҡиҲүйҖҷе№ҫзЁ®еҸҜиғҪжҖ§пјҢдёҰдё”жҠҠз•¶еүҚзҡ„еӯ—жҜҚзө„еҗҲеҠ е…ҘеҲ°жҡ«еӯҳи®ҠйҮҸдёӯгҖӮз•¶жҲ‘еҖ‘жһҡиҲүе®ҢжүҖжңүж•ёеӯ—жҷӮпјҢжҲ‘еҖ‘е°ұиғҪеҫ—еҲ°жүҖжңүеҸҜиғҪзҡ„еӯ—жҜҚзө„еҗҲгҖӮ

жҲ‘еҖ‘зҡ„зЁӢејҸжңғжӘўжҹҘжҲ‘еҖ‘жҳҜеҗҰе·Із¶“жһҡиҲүе®ҢдәҶжүҖжңүж•ёеӯ—пјҢеҰӮжһңжҳҜпјҢеүҮжҠҠз•¶еүҚеӯ—жҜҚзө„еҗҲеҠ е…ҘеҲ°зөҗжһңеҲ—иЎЁдёӯгҖӮеҰӮжһңйӮ„жңүеү©йӨҳж•ёеӯ—пјҢеүҮз№јзәҢйҒһиҝҙиҷ•зҗҶдёӢдёҖеҖӢж•ёеӯ—пјҢзӣҙеҲ°жүҖжңүеҸҜиғҪзҡ„зө„еҗҲйғҪиў«иҷ•зҗҶе®ҢгҖӮ

жңҖеҫҢпјҢеҮҪж•ёжңғеӣһеӮіеҢ…еҗ«жүҖжңүеҸҜиғҪеӯ—жҜҚзө„еҗҲзҡ„еҲ—иЎЁгҖӮ

еҜҰдҪң

def letterCombinations(self, digits: str) -> List[str]:

        digit_to_letter = {
            "2":'abc',
            "3":'def',
            "4":'ghi',
            "5":'jkl',
            "6":'mno',
            "7":'pqrs',
            "8":'tuv',
            "9":'wxyz'
        }

        def letterComs(letters, digits, tmpstr, tlen):
            if len(digits)==0:
                return
            
            if len(tmpstr)+1 == len(digits):
                for letter in digit_to_letter[digits[tlen]]:
                    letters.append(tmpstr+letter)
            else:
                for letter in digit_to_letter[digits[tlen]]:
                    letterComs(letters, digits, tmpstr+letter, tlen+1)
        
        letters = []
        letterComs(letters, digits, "", 0)
        return letters
