22. Generate Parentheses

и§ЈйЎҢжҖқи·Ҝ

зөҰе®ҡ n е°ҚжӢ¬иҷҹпјҢеҜ«дёҖеҖӢеҮҪж•ёдҫҶз”ҹжҲҗжүҖжңүеҪўжҲҗиүҜеҘҪжӢ¬иҷҹзҡ„зө„еҗҲгҖӮ

йҖҷеҖӢе•ҸйЎҢеҸҜд»ҘдҪҝз”ЁеӣһжәҜзҡ„ж–№жі•и§ЈжұәгҖӮжҲ‘еҖ‘дҪҝз”ЁдёҖеҖӢеҲ—иЎЁеӯҳе„ІжүҖжңүеҪўжҲҗиүҜеҘҪжӢ¬иҷҹзҡ„зө„еҗҲпјҢдёҰдҪҝз”ЁдёҖеҖӢйҒһжӯёеҮҪж•ёз”ҹжҲҗжүҖжңүзө„еҗҲгҖӮ

йҒһжӯёеҮҪж•ёйңҖиҰҒдёүеҖӢеҸғж•ёпјҡ

  • s жҳҜз•¶еүҚжӢ¬иҷҹзө„еҗҲпјӣ
  • left е’Ң right еҲҶеҲҘжҳҜз•¶еүҚй–Ӣе•ҹе’Ңй—ңй–үжӢ¬иҷҹзҡ„ж•ёйҮҸпјӣ еҰӮжһң len(s) == 2 * nпјҢеүҮд»ЈиЎЁжҲ‘еҖ‘е·Із¶“дҪҝз”ЁдәҶжүҖжңүзҡ„ n е°ҚжӢ¬иҷҹпјҢжүҖд»ҘжҲ‘еҖ‘е°ҮжӯӨзө„еҗҲж·»еҠ еҲ°еҲ—иЎЁдёӯгҖӮ

еҰӮжһң left < nпјҢеүҮд»ЈиЎЁжҲ‘еҖ‘еҸҜд»Ҙж·»еҠ й–Ӣе•ҹжӢ¬иҷҹгҖӮжүҖд»ҘпјҢжҲ‘еҖ‘иӘҝз”Ё backtrackпјҢеӮійҒһеҸғж•ё s + вҖҳ(вҖҳпјҢleft + 1 е’Ң rightгҖӮ

еҰӮжһң right < leftпјҢеүҮд»ЈиЎЁжҲ‘еҖ‘еҸҜд»Ҙж·»еҠ й—ңй–үжӢ¬иҷҹгҖӮжүҖд»ҘпјҢжҲ‘еҖ‘иӘҝз”Ё backtrackпјҢеӮійҒһеҸғж•ё s + вҖҳ)вҖҷпјҢleft е’Ң right + 1гҖӮ

жңҖеҫҢпјҢжҲ‘еҖ‘дёҚеӮід»»дҪ•еҸғж•ёиӘҝз”Ё backtrack д»Ҙй–Ӣе§Ӣз”ҹжҲҗзө„еҗҲгҖӮ

еҜҰдҪң

д»ҘдёӢжҳҜдёҖеҖӢдҪҝз”ЁеӣһжәҜжі•зҡ„ Python и§Јжұәж–№жЎҲпјҡ

def generateParenthesis(n):
    res = []
    def backtrack(s = '', left = 0, right = 0):
        if len(s) == 2 * n:
            res.append(s)
            return
        if left < n:
            backtrack(s + '(', left + 1, right)
        if right < left:
            backtrack(s + ')', left, right + 1)

    backtrack()
    return res

иӨҮйӣңеәҰ

жӯӨйЎҢзӣ®зҡ„жҷӮй–“иӨҮйӣңеәҰжҳҜ $O(4^n/\sqrt{n})$пјҢе…¶дёӯ $n$ жҳҜжӢ¬иҷҹе°Қж•ёгҖӮ

йҖҷжҳҜеӣ зӮәе°Қж–јжҜҸеҖӢжӢ¬иҷҹе°Қж•ёпјҢжңҖеӨҡжңү $2n$ еҖӢжӢ¬иҷҹпјҢеӣ жӯӨпјҢжңҖеӨҡжңү $4^n$ зЁ®зө„еҗҲж–№жЎҲгҖӮз„¶иҖҢпјҢз”ұж–јжңүдәӣзө„еҗҲдёҚеҗҲжі•пјҢеӣ жӯӨпјҢзңҹжӯЈзҡ„зө„еҗҲж•ёжҜ” $4^n$ е°ҸгҖӮж №ж“ҡ Catalan Number зҡ„иЁҲз®—пјҢзңҹжӯЈзҡ„зө„еҗҲж•ёжҳҜ $C_{n}^{2n} / (n+1)$пјҢеӣ жӯӨпјҢзңҹжӯЈзҡ„жҷӮй–“иӨҮйӣңеәҰжҳҜ $O(4^n/\sqrt{n})$гҖӮ
