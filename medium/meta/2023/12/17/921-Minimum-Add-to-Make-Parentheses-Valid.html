921. Minimum Add to Make Parentheses Valid

зөҰе®ҡдёҖеҖӢжӢ¬иҷҹеӯ—дёІsпјҢдёҖж¬Ўж“ҚдҪңеҸҜд»ҘеңЁеӯ—дёІзҡ„д»»дҪ•дҪҚзҪ®жҸ’е…ҘдёҖеҖӢжӢ¬иҷҹгҖӮи«ӢиЁҲз®—дҪҝеӯ—з¬ҰдёІжңүж•Ҳзҡ„жңҖе°Ҹж“ҚдҪңж¬Ўж•ёгҖӮ жңүж•ҲжӢ¬иҷҹеӯ—дёІзҡ„е®ҡзҫ©еҰӮдёӢпјҡ

  • е®ғжҳҜз©әеӯ—дёІпјҢ
  • е®ғеҸҜд»ҘеҜ«жҲҗABпјҲAиҲҮBйҖЈжҺҘпјүпјҢе…¶дёӯAе’ҢBйғҪжҳҜжңүж•Ҳеӯ—з¬ҰдёІпјҢжҲ–иҖ…
  • е®ғеҸҜд»ҘеҜ«жҲҗпјҲAпјүпјҢе…¶дёӯAжҳҜжңүж•Ҳеӯ—з¬ҰдёІгҖӮ

и§ЈйЎҢжҖқи·Ҝ

жҲ‘еҖ‘еҸҜд»ҘдҪҝз”Ёе Ҷз–ҠпјҲstackпјүзҡ„ж–№ејҸдҫҶи§ЈжұәйҖҷеҖӢе•ҸйЎҢгҖӮжҲ‘еҖ‘йҒҚжӯ·еӯ—дёІпјҢдёҰдҪҝз”ЁstackдҫҶиЁҳйҢ„жңӘеҢ№й…Қзҡ„е·ҰжӢ¬иҷҹзҡ„дҪҚзҪ®гҖӮз•¶йҒҮеҲ°еҸіжӢ¬иҷҹжҷӮпјҢжҲ‘еҖ‘жӘўжҹҘstackжҳҜеҗҰзӮәз©әпјҢеҰӮжһңдёҚжҳҜпјҢеүҮеҸҜд»ҘеҢ№й…ҚдёҖе°ҚжӢ¬иҷҹпјҢstack popгҖӮеҰӮжһңstackзӮәз©әпјҢеүҮйңҖиҰҒinsertдёҖеҖӢжӢ¬иҷҹгҖӮ

еҜҰдҪң

class Solution:
    def minAddToMakeValid(self, s: str) -> int:
        stack = 0          # з”Ёж–јиЁҳйҢ„жңӘеҢ№й…Қзҡ„е·ҰжӢ¬иҷҹж•ёйҮҸ
        add_left = 0  # з”Ёж–јиЁҳйҢ„йңҖиҰҒжҸ’е…Ҙзҡ„е·ҰжӢ¬иҷҹж•ёйҮҸ

        for char in s:
            if char == '(':
                stack += 1
            elif char == ')':
                if stack > 0:
                    stack -= 1
                else:
                    add_left += 1

        # жңӘеҢ№й…Қзҡ„е·ҰжӢ¬иҷҹж•ёйҮҸеҠ дёҠйңҖиҰҒжҸ’е…Ҙзҡ„еҸіжӢ¬иҷҹж•ёйҮҸеҚізӮәжңӘеҢ№й…Қзҡ„жӢ¬иҷҹзёҪж•ё
        return add_left + stack

йҒҚжӯ·дёҖж¬Ўеӯ—дёІsпјҢжҜҸж¬Ўж“ҚдҪңstackзҡ„жҷӮй–“иӨҮйӣңеәҰзӮә$O(1)$гҖӮеӣ жӯӨпјҢзёҪзҡ„жҷӮй–“иӨҮйӣңеәҰзӮә$O(n)$пјҢе…¶дёӯ$n$жҳҜеӯ—дёІsзҡ„й•·еәҰгҖӮ
